# Неделя 5: Контрольная приемка (mobile)

## Проверка сборки и тестов

- Команды:
  - `flutter pub get`
  - `flutter analyze`
  - `flutter test`
- Результат:
  - анализатор без ошибок
  - widget-тест проходит

## Проверенные сценарии

- Экран авторизации:
  - переключение режимов вход/регистрация
  - валидация обязательных полей
  - отображение ошибок backend
  - отображение состояния загрузки
- Интеграция с backend:
  - регистрация: `POST /auth/register`
  - логин: `POST /auth/login`
  - загрузка каталога: `GET /media-items`
- Роутинг:
  - неавторизованный пользователь видит auth-route
  - после успешного логина открывается app-shell
- Базовый UI:
  - экран библиотеки с пустым/ошибочным состояниями
  - экран профиля
  - переключение светлой/темной темы

- Week 5 закрыта на уровне MVP:
